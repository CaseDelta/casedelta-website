## SCOPE & POSITIONING NOTES

### What this video does NOT say
- Never names "AI" — CaseDelta is positioned as a **personal assistant**; the AI nature is implicit in the visuals.
- Never says "single screen" — uses "from one spot" instead (less diminutive).
- Never says "ask" or "prompt" — uses "tell Delta what to do" (assistant-coded, not LLM-coded).
- Never frames Delta as autonomous — the lawyer is always the subject of active verbs ("you make the calls," "tell Delta," "hand Delta").

### Core value prop expressed in the script
1. **Connection** — "CaseDelta connects every tool your firm uses" (Solution, line 3). Said once.
2. **Centralization** — "manage all of them, from one spot" (Solution, line 4). Said once.
3. **Simplicity** — "with a single sentence" (Solution); "Done all in one go" (Demo). Twice, in different framings.
4. **Personalization** — implicit in "in your style" omission (chose to demonstrate via visuals); explicit in "your new personal assistant" (Outro).
5. **Effortlessness / time gain** — "already handled for you before the day starts" (Dream); "So you can focus on winning more cases" (Dream resolution).

### Lawyer agency throughout
Every section has the lawyer as subject of an active verb:
- Dream — *winning more cases*
- Solution — *your firm*; *manage all of them*
- Demo — *Hand Delta…*; *Tell Delta…*
- Use Cases — *You make the calls*
- Outro — *the tools you already use* (your existing stack)
